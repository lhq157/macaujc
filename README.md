# 特码统计分析系统

基于历史数据的特别号统计分析工具，涵盖频率分布、结构偏差、遗漏周期、生肖分析、随机性检验等多维度统计检验，并自动生成专业分析报告。

> ⚠️ 本项目仅用于历史数据统计分析与学习研究，结果不具备任何预测能力，严禁用于赌博或选号。

---

## 项目结构

```
├── main.py          # 数据管道：拉取 → 解析 → 导出 CSV
├── report.py        # 一键报告生成器：分析 → HTML + PDF + Markdown
├── config.py        # 配置项（API、分析参数、输出目录）
├── analysis.ipynb   # 全部分析与可视化（Jupyter Notebook）
└── output/
    ├── YYYYMMDD.csv         # 按日期命名的数据文件，增量合并
    └── reports/
        ├── analysis_report.html  # 产品：自包含网页，直接分享
        ├── analysis_report.pdf   # 商品：专业排版，打印/发送
        └── README.md             # 说明书：项目内部文档
```

## 快速开始

### 环境要求

```bash
pip install numpy pandas matplotlib scipy ipywidgets jupyter
```

### 1. 拉取数据

```bash
python main.py
```

- 首次运行：全量拉取历史数据，写入 `output/YYYYMMDD.csv`
- 再次运行：与本地最新 CSV 增量合并，新文件包含全量历史

### 2. 一键生成报告

```bash
python report.py
```

自动完成：加载数据 → 全量计算 → 自动结论 → 输出三种格式

```
output/reports/analysis_report.html  ← 产品，直接发给别人
output/reports/analysis_report.pdf   ← 商品，打印/正式发送
output/reports/README.md             ← 说明书，项目内部文档
```

### 3. 深度分析（可选）

```bash
jupyter notebook analysis.ipynb
```

打开后执行 **Kernel → Restart & Run All**，或按需逐节运行。

## 数据说明

| 字段 | 说明 |
|------|------|
| `expect` | 期号 |
| `openTime` | 开奖时间 |
| `n1`~`n6` | 普通号码 |
| `special` | 特别号（1–49） |
| `zodiac` | 特别号对应生肖 |

CSV 文件以开奖日期命名（`YYYYMMDD.csv`），每次运行自动合并最新一期，即使数据源删除历史记录，本地档案也不会丢失。

## 分析模块

| 章节 | 内容 |
|------|------|
| 一、数据读取 | 自动加载本地 CSV，无数据时触发拉取 |
| 二、基础统计 | 号码频率、出现次数、变异系数 |
| 三、可视化 | 频次柱状图、7×7 热力图、奇偶/大小/尾数分布 |
| 四、生肖分析 | 频率检验 / 月度趋势 / 转移矩阵 / 区间交叉 / 间隔分析 / 奇偶结构 / 多窗口对比 / 滑动窗口交互台 |
| 五、间隔分析 | 平均间隔、当前遗漏、历史最长间隔 |
| 六、随机性检验 | 卡方检验、自相关检验 |
| 七、自动结论 | 多维度汇总，自动生成结构化分析报告 |
| 八、时间筛选 | 指定区间重跑全套分析 |
| 九、随机模拟对比 | 真实序列 vs 随机序列，KS检验证明统计上无法区分 |
| 十、异常检测 | 遗漏超阈值自动标红（2×警告 / 3.5×严重） |
| 十一、认知偏差分析 | 赌徒谬误 / 模式幻觉 / 幸存者偏差的数据证伪 |

### 4.9 滑动窗口交互台

固定窗口（30–500期）在全时间轴以步长=1滑动，逐窗口执行卡方检验，检验生肖分布的局部平稳性。

| 控件 | 说明 |
|------|------|
| 窗口大小 | 每次统计的期数（建议先用100） |
| 显著性 alpha | 异常判断门槛（默认0.05） |
| 趋势模式 | 单生肖 / Top5降噪 / 全部12肖 |
| 高亮异常区间 | 标红统计显著的时段 |

## 核心方法论

本系统基于以下统计学原理构建：

| 方法 | 应用章节 | 作用 |
|------|---------|------|
| 独立同分布（i.i.d.） | 全局前提 | 每期开奖为独立随机事件，历史数据对未来零预测力 |
| 卡方检验（χ²） | 二、四、六、九 | 验证号码/生肖是否偏离均匀分布 |
| KS 检验 | 九 | 检验真实序列与随机序列的分布差异 |
| 泊松过程 | 四、五 | 建模号码出现间隔，验证是否符合指数分布 |
| 时间序列平稳性 | 四 | 滑动窗口检验局部分布是否随时间漂移 |
| 自相关分析 | 六 | 检验相邻期之间是否存在序列依赖 |
| 大数定律 | 二、五 | 样本量越大，实测频率越趋近理论值 1/49 |
| 蒙特卡罗模拟 | 九、十一 | 用大规模随机模拟验证理论推导 |

> **核心结论**：所有统计检验均指向同一结论——特码序列符合离散均匀分布 i.i.d.，任何预测方法的准确率上限 = 随机猜测 = 1/49。
